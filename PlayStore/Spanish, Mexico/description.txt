Obtén una pantalla siempre encendida para tu teléfono o tableta.
Mantén tu pantalla encendida todo el tiempo.
¡La primera aplicación Always On de código abierto!

La idea es dar a los usuarios información acerca de la hora, fecha, notificaciones y más, sin tener que tocar el teléfono. Solo mirándola.

Esto es posible gracias a la tecnología de las pantallas AMOLED. La mayor parte de la pantalla se mantiene apagada, excepto algunos píxeles.

★Características especiales
• Pantalla siempre encendida
• Notificaciones - mira tus notificaciones sin tocar tu dispositivo
• Reglas automáticas - ahorra batería utilizando reglas predefinidas
• Movimiento automático - evita quemadura de la pantalla AMOLED
• Modo nocturno automático - oscurece la pantalla automáticamente en ambientes oscuros
• Modo bolsillo - bloquea el dispositivo cuando lo dejas en tu bolsillo, para ahorrar batería
• Dos toques para desbloquear + deslizar arriba para desbloquear + teclas de volumen para desbloquear + botón de volver para desbloquear
• Forzar orientación - elige la orientación de pantalla que prefieras
• Personalización - cambia el color del texto, su tamaño, brillo y más
• ¡GRATIS Y DE CÓDIGO ABIERTO!

Antes de usar esta aplicación, habilita el permiso para "superponer esta aplicacion sobre otras".

La aplicación está en fase BETA, todavía tiene errores y carece de funciones.

Permisos
<b>Permisos de Cámara</b>
La aplicación necesita permiso de cámara para reconocer su apertura y desactivar Always On.
<b>Permiso de acceso al teléfono</b>
La aplicación necesita permisos de acceso al teléfono para reconocer llamadas entrantes, ocultar Always On y mostrar la llamada.
<b>Modificar configuración de permisos del sistema</b>
La aplicación necesita permiso para modificar la configuración del sistema, para poder cambiar el brillo de la pantalla de bloqueo.
